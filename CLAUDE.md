# Проект: сайт-визитка grabdata.dev (статическая версия)

## Что это
Боевой одностраничный сайт-визитка Аллы Станчевой (Junior Data Analyst, Amsterdam).
Чистая статика — HTML + CSS, без сборки. Хостинг — собственный сервер Hetzner (`main`),
nginx + Let's Encrypt, домен **grabdata.dev**.

## Технологии
- Чистый HTML5 + CSS (без фреймворков, без JS-сборки)
- Один внешний ресурс: шрифт Inter через Bunny Fonts (EU-хостинг, GDPR-safe — не Google Fonts)
- Контактное окно (`<dialog>`) с маленьким inline-скриптом для формирования `mailto:`

## Структура
- `index.html` — единственная страница (Hero, Projects, Skills, About, Values)
- `styles.css` — стили, переменные в `:root`
- `images/cover.jpg`, `favicon.svg` — ассеты

## Правила работы
- Язык общения — русский
- Перед коммитом проверяй файлы на захардкоженные секреты (на этом сайте их быть не должно)
- Не усложнять: это намеренно простая статика, которую владелец правит руками
- Деплой — через GitHub: `git push` → на сервере `@sysadmin` делает `git pull`

## Происхождение
Дизайн воспроизведён из Astro-проекта `сайт_визитка_Алла_Станчева_2` (соседняя папка).
Версии независимы — правки дизайна переносятся вручную.
